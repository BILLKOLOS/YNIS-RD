# Images Folder

This folder contains all images used in the YNIS-RD website.

## Required Images

Please upload the following images to this folder:

### Hero Section
- **hero-background.jpg** - Main hero background image (1920x1080px recommended)
  - Should feature youth, community engagement, or related imagery
  - Will be used with overlay, so contrast is important

### Programs
- **children-rights.jpg** - Children rights program image (800x600px recommended)
  - Should show children in educational or community settings
  
- **environment.jpg** - Environment program image (800x600px recommended)
  - Should show environmental activities, tree planting, or nature conservation
  
- **reproductive-health.jpg** - Reproductive health program image (800x600px recommended)
  - Should show health education or community health activities

### Team Photos
- **ceo-photo.jpg** - Photo of Dr. Samson Barongo (500x500px recommended)
  - Professional headshot, square format
  
- **team-ceo.jpg** - Same as ceo-photo.jpg (can be duplicate)
  
- **team-programme.jpg** - Programme Officer photo (500x500px recommended)
  - Professional headshot, square format
  
- **team-field.jpg** - Field Officers photo (500x500px recommended)
  - Can be a group photo or representative image

## Image Guidelines

- **Format**: JPG or PNG
- **Quality**: High resolution, but optimized for web (under 500KB each)
- **Orientation**: 
  - Hero: Landscape (16:9)
  - Programs: Landscape (4:3)
  - Team: Square (1:1)
- **Content**: Professional, warm, and representative of YNIS-RD's mission

## Fallback

If you don't have specific images yet, you can use:
- Stock photos from free sources like Unsplash or Pexels
- Placeholder images with similar dimensions
- Professional photos taken at YNIS-RD events and activities
